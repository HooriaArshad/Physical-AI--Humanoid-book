# Docasuros Frontend

React + TypeScript + Vite frontend for Physical AI & Humanoid Robotics.

**Created by: Hooria Arshad**

## Setup

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
