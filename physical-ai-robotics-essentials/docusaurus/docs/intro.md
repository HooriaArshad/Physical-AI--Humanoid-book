## Module 1: The Robotic Nervous System (ROS 2)

### Overview

ROS 2 (Robot Operating System 2) serves as the **middleware backbone** for modern robotics. It provides the communication infrastructure that connects sensors, actuators, and AI systems into a cohesive robotic platform.

:::tip Think of it this way
ROS 2 is like the **nervous system** that carries signals between the robot's brain and body.
:::

### What You'll Learn

| Topic | Description |
|-------|-------------|
| **ROS 2 Nodes** | Creating, launching, and managing independent computation units |
| **Topics & Messaging** | Publish/subscribe pattern for sensor data and control commands |
| **Services & Actions** | Synchronous and asynchronous request-response mechanisms |
| **Python Integration** | Bridging AI agents to ROS controllers via `rclpy` |
| **URDF** | Defining humanoid robot structure and joints |
| **TF2 Transform Library** | Tracking relative positions of robot links in 3D space |
